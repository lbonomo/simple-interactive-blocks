An interactive block with the Interactivity API.

== Description ==
This plugin provides a set of easy-to-use blocks that add interactive elements to your WordPress site. Currently includes:

 - Simple Interactivity Search: Offers live, real-time search suggestions as users type, using the WordPress REST API for efficient performance without relying on AJAX.
