# Simple interactive blocks

## Developer workflow
 - Start server
   `npm run env start`
 - Watching changes.
   `npm start`

## Create some post
```bash
npm run env run cli /var/www/html/shared/posts.sh
```

To enter to docker container.
```bash
npm run env run cli bash
```

```bash
npm run env logs all --watch
```
